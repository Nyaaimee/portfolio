HTML

HTML stands for Hyper Text Markup language and it is a standard markup language for documents designed to be displayed in a web browser. In a more simple term i can say it's a human skeletun that has no flesh and needs to be fed.

CSS

CSS stands for Cascading stylesheet is a style sheet language used for describing the presentation of a document writen in a markup language such a as html. With css i can beautify my Website as i want with the different css styles adding flesh to my HTML.

JavaScript

Also abbreviated as js is a language used by programmers across the world to create dynamic and interractive websites. It adds more functionality to a website which without it html and css are inactive.

Vuejs

Vuejs is a powerful front-end JavaScript framework used to build single page application. Single page application means a single html page is loaded from the server and routes all request to the server back to the html page itself. It is cool because it contains both the html,css and js all inone page.
